# Online-Retail-Sales-Analysis
This project analyzes an online retail sales dataset to uncover key business insights related to revenue performance, customer behavior, product categories, and payment preferences.


📌 Project Overview

This project analyzes an online retail sales dataset to uncover key business insights related to revenue performance, customer behavior, product categories, and payment preferences.

The goal of this project was to simulate a real-world Data Analyst role by performing:

Data Cleaning (Python – Pandas)

Advanced SQL Analysis

Statistical Analysis

Dashboard Development (Power BI)

Business Storytelling

This project demonstrates end-to-end analytical thinking — from raw data to actionable business insights.

🛠 Tools & Technologies Used

Python (Pandas, NumPy) – Data cleaning and preprocessing

MySQL – Advanced SQL queries and aggregation

Power BI – Dashboard creation and visualization

Statistics – Mean, Median, Standard Deviation, Correlation

📂 Dataset Features

The dataset includes:

Order ID

Order Date

Customer ID

Customer City

Product Category

Product Name

Quantity

Price

Discount

Payment Method

🧹 Data Cleaning Process

Removed missing values

Eliminated duplicate records

Converted date columns into proper datetime format

Created calculated Revenue column

Validated discount and pricing structure

🧮 SQL Analysis Performed

Total Revenue Calculation

Monthly Sales Trend Analysis

Top 5 Customers by Revenue

Category-wise Revenue Contribution

Payment Method Distribution

Advanced SQL concepts used:

GROUP BY

ORDER BY

LIMIT

Aggregate Functions

Calculated fields

📊 Statistical Analysis

Mean Order Value

Median Order Value

Standard Deviation of Revenue

Correlation between Quantity, Discount, and Revenue

This helped understand revenue stability and customer purchasing patterns.

📈 Power BI Dashboard Features

The interactive dashboard includes:

KPI Cards (Total Revenue, Total Orders, Average Order Value)

Revenue Trend Line Chart

Category vs Revenue Bar Chart

Payment Method Distribution Pie Chart

Top 5 Customers Table

The dashboard was designed with business storytelling principles to communicate insights clearly.

🔎 Key Business Insights

Revenue is concentrated in top-performing product categories.

Customer purchasing behavior varies by payment method.

A small group of high-value customers contributes significantly to total revenue.

Monthly revenue trends indicate potential seasonal patterns.
